# 2026-04-20

## map.html 模拟机兼容性大修

**核心问题**：模拟机 WebView 对 `fetch()` API 支持极差，所有网络请求几乎都发不出去，导致骑行和公交路线规划全部失败。

**修复内容**：
1. 所有 `fetch()` 替换为 `XMLHttpRequest`（封装为 xhrGet()，12秒超时）
2. 删除骑行 tab（AMap bicycling API 根本不存在/未授权）
3. 公交 API 的 city 参数从 `city=beijing` 改为 `city=北京`
4. 简化 CSS：移除渐变、box-shadow 等模拟机可能不支持的特性
5. 所有 forEach 替换为普通 for 循环（兼容性更好）
6. 提交 b90e01f，已 push 到 remote
